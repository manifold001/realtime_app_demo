# Real time app implementation using long polling and web sockets

## Usage:

Install dependencies:
```
npm install
```

### Long polling version

Run the web server 
```bash
node long_poll_server.js
```

Run the web client 
```bash
node long_poll_client.js
```

Post some data
```
./request.sh
```

### Websocket version

Run the web server 
```bash
node websocket_server.js
```

Run the web client 
```bash
node websocket_client.js
```

Post some data
```
./request.sh
```
